# java-lotto-precourse

## 프리코스 3 주차 - 로또

### 기능 목록

- [ ] 로또
    - [ ] 6개의 중복 없는 랜덤 숫자 뽑기 (1~45)
    - [ ] 일치하는 번호 개수 확인
    - [ ] 수익률 계산
- [ ] 입력
    - [ ] 구입 금액 입력
    - [ ] 당첨 로또 번호 6개와 보너스 번호 1개 입력
- [ ] 출력
    - [ ] 로또 안내 문구 출력
    - [ ] 당첨 통계 출력
- [ ] 예외 처리
    - [ ] 금액 입력 예외 처리
